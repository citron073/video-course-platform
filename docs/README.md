# ドキュメント目次

動画講座プラットフォームの設計・進捗・運用をここで管理する。

| ファイル | 用途 |
|---|---|
| [実装プラン.md](./実装プラン.md) | 確定した要件と設計（壁打ちで決めた方針の正本） |
| [進捗記録.md](./進捗記録.md) | 何をどこまで作ったかの記録・教訓ログ |
| [未実行タスク.md](./未実行タスク.md) | 残タスクと再開手順（次に何をやるか） |
| [運用手順.md](./運用手順.md) | レッスン追加・seed・デプロイの手順書 |
| [テスト手順.md](./テスト手順.md) | ユニットテストの使い方・対象・追加方法 |
| [横展開ガイド.md](./横展開ガイド.md) | 指示→実行の対応表・再利用テンプレ・立ち上げチェックリスト |

> タスクのチェックリスト（台帳）は プロジェクト直下の [`../Tasks.md`](../Tasks.md)。
> 作業ルール（着手前にdocs確認・完了後にTasks.md更新・テストの鉄則・DoD）は [`../CLAUDE.md`](../CLAUDE.md)。

> プロジェクトのセットアップ手順は親フォルダの [`../README.md`](../README.md) を参照。
> このフォルダは「設計と記録」、`../README.md` は「使い方の入口」という分担。
