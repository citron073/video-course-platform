## 優先度: 高（公開までに必須）

### 1. 動画IDの差し替え
現在はダミーID（`dQw4w9WgXcQ`）。
- 手順: `db/seed.ts` の `SECTIONS[].lessons[].youtubeId` を実際の限定公開動画IDへ → `npm run seed`

### 2. 講座タイトル・説明・章立ての確定
- 手順: `db/seed.ts` の `COURSE`（slug/title/description）と `SECTIONS`（章とレッスン）を実文言へ → `npm run seed`
- `COURSE.slug` を変えたら `lib/config.ts` の `COURSE_SLUG` も合わせる。

### 3. Vercelデプロイ
- 手順（詳細は [運用手順.md](./運用手順.md)）:
  1. `turso db create video-course`（CLIログインが必要・たにさん実行）
  2. URL とトークンを取得し、ローカル `.env.local` を本番値にして `npm run db:push` → `npm run seed`
  3. Vercel の環境変数に `TURSO_DATABASE_URL` / `TURSO_AUTH_TOKEN` を登録
  4. デプロイ後、ライブで動画再生まで確認
