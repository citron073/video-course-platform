## 教訓ログ（ハマり→解決）

- **ディレクトリ名が日本語（`講座用`）で `create-next-app .` が使えない**
  → パッケージ名検証で弾かれるため、手動スキャフォールド（`package.json` 自作 + `npm install @latest`）で構築。
- **seed が `TURSO_DATABASE_URL 未設定` で落ちる**
  → ESMの import ホイスティングで dotenv の `config()` より先に `db/index.ts` が評価されるのが原因。
     `db` を `main()` 内で動的 import（`await import("./index")`）して解決。
- **サイドバーの現在地ハイライト**
  → App Router のレイアウトは子セグメントの param を取れない。`LessonSidebar` を client 化し `usePathname` で判定。
